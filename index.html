Projects

Selected projects and consulting work.

libplacebo

Author & Maintainer

Reusable library for GPU-accelerated image and video processing primitives, including a comprehensive rendering pipeline. Supports Vulkan, OpenGL, Metal (via MoltenVK), and Direct3D 11. Features advanced color management, HDR tone mapping algorithms, and high-quality image scaling. Author of ~95% of the codebase.

Vulkan OpenGL Color Management HDR

libswscale Rewrite

FFmpeg Core Component ¬∑ Funded by Sovereign Tech Fund

Complete overhaul of libswscale, a widely used FFmpeg component for pixel format conversion. Developed a novel approach using a domain-specific embedded pseudo-language to generate custom conversion routines. The system interprets conversion logic and links hand-written SIMD routines, achieving dramatic performance improvements across all supported formats.

FFmpeg SIMD DSL Performance

checkasm Overhaul

Testing & Benchmarking Infrastructure

Major overhaul of checkasm, a helper library for verifying correctness and benchmarking SIMD routines in open source projects. Improved portability, user friendliness, and made it reusable across projects. Enhanced benchmark statistics collection and analysis, enabling cycle-accurate performance metrics for large projects in seconds rather than minutes.

Testing SIMD Benchmarking Tooling

FFmpeg Contributions

Architectural Improvements & Optimization

Ongoing contributions focused on architectural improvements, including better format negotiation (obsoleting YUVJ formats), filter optimization, and Vulkan enhancements. Work spans color management, performance optimization, and modernization of core components.

FFmpeg Vulkan Architecture Optimization
